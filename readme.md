 GNN Path Predictor

A beginner-friendly Graph Neural Network (GNN) project that predicts paths and vulnerabilities in a simulated network graph.
Built with PyTorch Geometric and Streamlit, this app helps visualize and understand how GNNs learn relationships between connected nodes.

What It Does

Generates a simulated network graph (servers, routers, users, etc.)

Uses a GNN model to predict possible vulnerable paths

Displays predictions and probabilities visually via Streamlit
